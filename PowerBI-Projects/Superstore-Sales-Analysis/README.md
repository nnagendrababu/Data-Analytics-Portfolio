# Superstore Sales Performance Dashboard

## Project Overview

This project analyzes retail sales performance using the Superstore dataset in Power BI.

The dashboard provides insights into sales, profit, customer performance, product performance, and regional trends.

---

## Tools Used

- Power BI
- SQL
- Excel

## Dashboard Features

### Sales Analysis
- Sales by Region
- Monthly Sales Trend

### Profit Analysis
- Profit by Category

### Customer Analysis
- Top 10 Customers by Sales

### Product Analysis
- Top 10 Products by Sales

### Interactive Filters
- Region Slicer

---

## Key Business Insights

### Regional Performance
- West region generated the highest sales.
- South region generated the lowest sales.

### Category Performance
- Technology category produced the highest profit.
- Furniture category produced the lowest profit.

### Sales Trend
- Sales showed an upward trend toward the later periods.

### Customer Insights
- A small number of customers contribute a significant portion of revenue.

### Product Insights
- Top-selling products contribute disproportionately to total sales.
